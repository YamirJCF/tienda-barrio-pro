# Discussions

Esta carpeta contiene documentos con observaciones o conflictos por resolver.

## Nomenclatura

- `obs-ux-nombre.md` - Observación de UX
- `obs-db-nombre.md` - Observación de Base de Datos
- `obs-lead-nombre.md` - Observación del Lead

## Flujo

1. Documento llega de `proposals/` con conflicto
2. Se resuelve la discusión
3. Documento aprobado → mover a raíz de `01_REQUIREMENTS/`
