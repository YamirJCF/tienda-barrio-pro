# Proposals

Esta carpeta contiene borradores y propuestas de nuevos requerimientos.

## Nomenclatura

- `prop-nombre-v1.md` - Propuesta inicial
- `prop-nombre-v2.md` - Revisión

## Flujo

1. Crear archivo aquí
2. Solicitar revisión al Lead
3. Si hay conflicto → mover a `discussions/`
4. Si aprobado → mover a raíz de `01_REQUIREMENTS/`
