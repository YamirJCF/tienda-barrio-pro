Preguntas Rápidas

¿Qué hago si se va el internet? expand_more

No te preocupes. La app guarda tus ventas localmente. Cuando vuelva la conexión, se sincronizarán automáticamente.

¿Cómo agrego un nuevo producto? expand_more

Ve a la sección "Inventario" y pulsa el botón "+" azul. Puedes escanear el código de barras o ingresarlo manualmente.

¿Puedo tener dos cajas abiertas? expand_more

Sí, pero necesitas permisos de administrador. Ve a "Mi Equipo" para configurar los permisos de cada usuario.

¿Cómo elimino una venta errónea? expand_more

Desde el historial de ventas, selecciona la transacción y pulsa "Anular". Necesitarás ingresar el motivo.
